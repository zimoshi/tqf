## Installation

Clone this repository:

```bash
git clone https://github.com/YOUR-USERNAME/TQF.git
cd TQF
```

(Optional: if you want to pip install locally)

```bash
pip install .
```

---

## Usage

```bash
python3 tqf.py
```

It will launch the animated TQF boot screen, then start the quiz engine.
